## **I. Git basic:**
- Version control system
- 3 vùng làm việc: working directory, staging, repository
- Các câu lệnh git: 
    - git init
    - git add .
    - git commit -m"message"
    - git status
    - git log
    - git config user.name ""
    - git config user.email ""
